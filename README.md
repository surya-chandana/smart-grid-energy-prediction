# Smart Grid Energy Prediction

## Project Description
This project predicts the stability of a smart grid using Machine Learning and provides predictions through a Streamlit web application.

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

## Dataset
Smart Grid Stability Augmented Dataset

## How to Run
1. Install the required libraries.
2. Run:
   streamlit run app.py
